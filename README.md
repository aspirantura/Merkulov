# 🩺 Алгоритм выбора хирургической тактики при злокачественных новообразованиях почек у детей

Электронный алгоритм поддержки принятия клинических решений для детских онкохирургов
при планировании хирургического лечения злокачественных новообразований почек у детей.

## 🎯 Возможности

- Определение оптимального типа операции (нефрэктомия / резекция почки)
- Определение оптимального хирургического доступа (лапароскопический / открытый)
- Обоснование рекомендации с указанием факторов «за» и «против»
- Предостережения и требования к выполнению вмешательства
- Схематичная визуализация почки с локализацией опухоли
- Автоматический расчёт площади поверхности тела (ППТ)

## 📊 Основные критерии

### Лапароскопическая нефрэктомия
- Объём опухоли после ПХТ до **200 см³**
- Допустимы: периферическая локализация, кистозный компонент, слабый ответ на ПХТ

### Лапароскопическая резекция почки
- Периферическая (полюсная) локализация
- Объём опухоли до **100 см³**
- Сохранение ≥66% паренхимы
- Отступ ≥5 мм от края опухоли

### Абсолютные противопоказания к лапароскопии
- Опухолевый тромбоз
- Инвазия в магистральные сосуды
- Билатеральное поражение (для нефрэктомии)
